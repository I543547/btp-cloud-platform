# Read Clones

Read a clone job. The entity is only available after the job has finished. To track the status of the job, you can use the GET /Pull\(guid’UUID’\) request.



## Request

**URI:** /sap/opu/odata/sap/MANAGE\_GIT\_REPOSITORY/Clones

**Operation Type:** CRUD

**HTTP Method:**GET
